# Setup
1. Create static bundle
```
npm run build:dev
```
2. Create virtual environment
```
npm run create-venv
```
3. Activate virtual environment
```
source env/bin/activate
```
4. Setup environment variables
```
npm run setup
```
5. Initiate app
```
npm start
```
6. Deactivate virtual environment when done
```
deactivate
```
