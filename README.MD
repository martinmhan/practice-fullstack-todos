# Setup
1. Create virtual environment
```
npm run create-venv
```
2. Activate virtual environment
```
source env/bin/activate
```
3. Setup environment variables
```
npm run setup
```
4. Create static bundle
```
npm run build:dev
```
5. Initiate app
```
npm start
```
6. Deactivate virtual environment when done
```
deactivate
```
